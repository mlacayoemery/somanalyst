Database File Merge

uiDBFmerge.dbfmerge(ifile1, ifile2, ofile)

Creates a single DBF file that contains both input DBF files.

Arguments:
ifile1

The first input DBF file.

ifile2

The second input DBF file.

ofile

The output DBF file.
